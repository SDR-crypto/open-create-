Open Create - Container Storage Guide

Introduction

Open Create - Container Storage is an advanced system designed to manage large volumes of data efficiently and securely. This guide will help you understand how to create and manage container storage using the Open Create library.

Installation

To install Open Create - Container Storage, use the following command:

npm install container-storage

Creating a Container

To create a container, you need to initialize the ContainerStorage class and use the createContainer method. Here is an example:

const { ContainerStorage } = require('container-storage');

const storage = new ContainerStorage();
storage.createContainer("images");

Storing Data

To store data in a container, use the storeData method. Here is an example:

storage.storeData("images", "image1", "data_of_image1");

Retrieving Data

To retrieve data from a container, use the retrieveData method. Here is an example:

console.log(storage.retrieveData("images", "image1"));  // Output: { data: 'data_of_image1', timestamp: '2025-03-02T19:56:37.000Z' }

Deleting a Container

To delete a container, use the deleteContainer method. Here is an example:

storage.deleteContainer("images");

Listing All Containers

To list all containers, use the listContainers method. Here is an example:

console.log(storage.listContainers());  // Output: ['images']

Listing Keys in a Container

To list all keys in a container, use the listContainerKeys method. Here is an example:

console.log(storage.listContainerKeys("images"));  // Output: ['image1']

Checking Container Existence

To check if a container exists, use the containerExists method. Here is an example:

console.log(storage.containerExists("images"));  // Output: true

Checking Key Existence in a Container

To check if a key exists in a container, use the keyExists method. Here is an example:

console.log(storage.keyExists("images", "image1"));  // Output: true
